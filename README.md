# 📊 USA Census ETL Pipeline (2023)

¡Bienvenido a mi proyecto de análisis de datos! Este es un sistema **ETL** completo que automatiza la obtención, limpieza y visualización de la población de Estados Unidos.

## 🚀 ¿Qué hace este proyecto?
Este pipeline realiza tres pasos críticos:
1.  **Extracción & Limpieza (Python):** Toma datos crudos del Censo de EE. UU. (CSV), limpia los nombres de los estados y filtra la información relevante usando la librería `Pandas`.
2.  **Carga (MySQL):** Los datos procesados se inyectan automáticamente en una base de datos relacional para su almacenamiento seguro y estructurado.
3.  **Visualización (Power BI & Matplotlib):** * Genera gráficos estadísticos con `Seaborn` directamente desde Python.
    * Se conecta a Power BI para crear dashboards interactivos y mapas de calor.

## 🛠️ Tecnologías utilizadas
* **Lenguaje:** Python 3.x
* **Librerías:** Pandas, Matplotlib, Seaborn, SQLAlchemy.
* **Base de Datos:** MySQL.
* **BI:** Power BI Desktop.

## 📈 Resultado Final
El proyecto logra visualizar el ranking de los 10 estados más poblados en 2023, destacando a California, Texas y Florida con datos precisos y formateados.
